# pycharm 快捷键

CTRL + P       将光标放在函数的圆括号里，按CTRL + P ,可以显示参数

CTRL + ]        光标跳到行尾

CTRL + [        光标跳到行首

SHIFT + Enter  快速 换行，即光标无需在行尾

CTRL + /        单行注释

''' '''         块注释 
CTRL + Alt +L  自动排版对齐

CTRL + Shift + F 全局搜索

## Linux 快捷键

复制文件 cp file_a file_b
 命令行中回到行首 行尾 ctrl + a (ahead)/ e(end)
